# OJK Portal — UI Kit

Interactive prototype of the OJK public-facing web portal. Built with React, styled using the shared `colors_and_type.css` design tokens.

## Components

| Component | Description |
|-----------|-------------|
| `Navbar` | Primary navigation bar (red background, white text, nav links) |
| `Hero` | Hero section with headline, description, dual CTA buttons |
| `StatsBar` | 4-column statistics strip |
| `ServiceCard` | Service area card with badge, title, description, link |
| `Alert` | Info/warning alert banners |
| `DataTable` | Regulation data table with status badges |
| `Footer` | Dark footer with links |

## Usage

Open `index.html` in a browser. All components are in `App.jsx`.
